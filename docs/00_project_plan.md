** 기능
1. 매수 후 +(3,5,7,10)% 익절 주문 버튼

		버튼 종류:
		- [매수 후 +3% 익절]
		- [매수 후 +5% 익절]
		- [매수 후 +7% 익절]
		- [매수 후 +10% 익절]

		동작 순서:
		a. 선택 종목 현재가 확인
		b. 동일 종목의 기존 미체결 매수 주문이 있으면 취소
		c. 미체결 매수 주문 취소 완료 확인
		d. 현재가/시장가 매수 주문 전송
		e. 체결 이벤트 대기
		f. 실제 체결가, 체결수량 확인
		g. 체결가 기준 선택 익절률(+3% / +5% / +7% / +10%) 계산
		h. 호가 단위에 맞게 익절가 보정
		i. 체결수량 기준 익절 매도 주문 자동 등록

		예)
		매수가 10,000원 체결
		+5% 익절 버튼 선택
		→ 익절 매도 주문 10,500원 자동 등록

		j. 매수 체결 후 30초 타이머 시작
		k. 익절 매도 주문이 체결 완료되지 않은 상태에서 현재가 모니터링
		l. 현재 수익률이 -5% 이하이면 방어 매도 조건 발생
		m. 기존 익절 매도 주문 취소
		n. 취소 완료 확인
		o. 보유수량 기준 방어 매도 실행
		
	2. 방어 버튼(반자동) - 현재가 모니터링만 수행하는 방어 모드
		전제 조건:
		- 아직 익절 매도 주문이 체결 완료되지 않은 상태
		- 보유수량이 존재하는 상태
		- 매도 주문/취소 처리 중이 아닌 상태

		동작:
		a. 방어 버튼 클릭 시 방어 모드 진입
		b. 실제 매도 주문은 즉시 발생하지 않음
		c. 누를 때마다 현재가 기준 -1%씩 방어 예상 매도가를 화면에 노출
		   - 1회 클릭: 현재가
		   - 2회 클릭: 현재가 -1%
		   - 3회 클릭: 현재가 -2%
		   - 4회 클릭: 현재가 -3%
		d. 사용자가 [확인] 버튼 클릭 시
		   - 기존 익절 매도 주문 취소
		   - 취소 완료 확인
		   - 보유수량 기준 방어 매도 주문 실행
		e. 사용자가 [취소] 버튼 클릭 시
		   - 방어 모드 해제
		   - 기존 익절 매도 주문은 유지
		

	3. 최종 즉시 현재가 매도 버튼[전량 즉시 매도]

** 필수 안전 장치

		1. 1일 최대 매매금액 제한

		   * 하루 동안 매수 가능한 총 금액을 제한한다.
		   * 설정 금액 초과 시 추가 매수 주문을 차단한다.

		2. 종목당 최대 매수금액 제한

		   * 한 종목에 투입 가능한 최대 금액을 제한한다.
		   * 동일 종목 중복 진입 및 과매수를 방지한다.

		3. 중복 매수 방지

		   * 동일 종목에 매수 주문 중이거나 보유 중인 경우 추가 매수를 제한한다.
		   * 기존 미체결 매수 주문이 있는 경우 취소 완료 후 새 주문을 전송한다.

		4. 주문 실패 시 재시도 제한

		   * 주문 실패 시 무한 재시도를 방지한다.
		   * 재시도 횟수를 제한하고, 실패 사유를 로그로 저장한다.

		5. 실계좌 / 모의투자 모드 구분

		   * 프로그램 실행 시 현재 모드가 실계좌인지 모의투자인지 명확히 표시한다.
		   * 실계좌 모드에서는 주문 전 경고 표시 또는 확인 절차를 둔다.

		6. 로그 저장

		   * 매수 주문, 매도 주문, 취소 요청, 체결 이벤트, 오류 발생 내역을 저장한다.
		   * 주문번호, 종목코드, 주문수량, 주문가, 체결가, 체결수량, 시간, 실패 사유를 기록한다.

		7. 프로그램 종료 시 미체결 주문 확인

		   * 프로그램 종료 전 미체결 주문 존재 여부를 확인한다.
		   * 미체결 주문이 있으면 사용자에게 알림을 표시한다.
		   * 필요 시 사용자가 직접 취소 또는 유지 여부를 선택할 수 있게 한다.

		8. 종목별 JSON 파일 설정값 관리

		   * 종목별 매수금액, 익절률, 손절률, 방어 조건, 최대 매수금액 등을 JSON 파일로 관리한다.
		   * 프로그램 시작 시 설정값을 로드하고, 변경 시 저장한다.

		9. 호가 단위 보정

		   * 익절가와 손절가 계산 시 주문 가능한 호가 단위에 맞게 가격을 보정한다.
		   * 익절가는 체결 가능성을 고려해 호가 단위에 맞게 보정한다.
		   * 손절가와 방어 매도가도 호가 단위에 맞게 보정한다.

		10. 기존 익절 주문 취소 후 매도 실행

		* 방어 매도 또는 즉시 전량 매도 실행 전 기존 익절 매도 주문을 먼저 취소한다.
		* 취소 완료 확인 후 보유수량 기준으로 매도 주문을 전송한다.

		11. 버튼 중복 클릭 방지

		* 주문 처리 중에는 관련 버튼을 비활성화한다.
		* 더블클릭 또는 연속 클릭으로 인한 중복 주문을 방지한다.

		12. 보유수량 재확인

		* 매도 주문 전 항상 실제 보유수량을 다시 확인한다.
		* 체결수량과 보유수량이 다를 경우 실제 보유수량 기준으로 매도한다.

		- 부가 기능

		1. 계좌 잔고 조회

		   * 현재 보유 종목, 보유수량, 평균매수가, 평가금액, 평가손익을 조회한다.

		2. 미체결 주문 조회

		   * 현재 미체결 매수/매도 주문을 조회한다.
		   * 종목별 주문 상태를 화면에 표시한다.

		3. 오늘 체결 내역 조회

		   * 당일 매수/매도 체결 내역을 조회한다.
		   * 체결가, 체결수량, 체결시간을 확인한다.

		4. 종목별 상태 복구

		   * 프로그램 시작 시 계좌 잔고, 미체결 주문, 오늘 체결 내역을 기준으로 종목별 상태를 복구한다.
		   * 보유 중인 종목, 익절 주문이 걸린 종목, 미체결 주문이 있는 종목을 구분한다.

		* 버튼 비활성화 조건

		1. 매수 주문 중

		   * 매수 버튼 비활성화
		   * 동일 종목 중복 매수 방지

		2. 매도 주문 중

		   * 매도 버튼 비활성화
		   * 방어 버튼 비활성화
		   * 즉시 전량 매도 버튼 비활성화

		3. 취소 처리 중

		   * 방어 버튼 비활성화
		   * 즉시 전량 매도 버튼 비활성화
		   * 기존 익절 주문 취소가 완료될 때까지 추가 매도 명령을 차단한다.

		4. 익절 주문 처리 중

		   * 익절 주문 등록 완료 전까지 방어 버튼 비활성화
		   * 익절 주문 상태가 명확해진 후 방어 모드 진입 가능

		5. 보유수량 없음

		   * 방어 버튼 비활성화
		   * 즉시 전량 매도 버튼 비활성화
