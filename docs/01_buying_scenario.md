# KRX / NXT / 장전 동시호가 / API 호출 전략 정리

## 1. 기본 개념

KRX는 기존 한국거래소이고, 일반적으로 말하는 정규장은 KRX 기준이다.

KRX 정규장:
09:00 ~ 15:30

KRX 장전 동시호가:
08:30 ~ 09:00

KRX 시초가 결정:
09:00

NXT는 넥스트레이드라는 대체거래소다.

NXT 거래시간:
프리마켓   08:00 ~ 08:50
메인마켓   09:00:30 ~ 15:20
애프터마켓 15:40 ~ 20:00

NXT 프리마켓 가격이 KRX 정규장 시초가가 되는 것은 아니다.

NXT 프리마켓 마지막 체결가 = 참고 가격
KRX 정규장 시초가 = KRX 장전 동시호가에서 결정

예시:
전일 종가: 300,000원
NXT 프리마켓 마지막 체결가: 306,000원
KRX 장전 동시호가 예상체결가: 304,000원
09:00 실제 시초가: 304,000원

즉, 프리마켓 가격은 참고용이고, 실제 정규장 시작 가격은 KRX 동시호가에서 결정된다.


## 2. 장전 동시호가 개념

장전 동시호가는 주문이 바로 체결되는 방식이 아니다.

08:30 ~ 09:00 동안 매수 주문과 매도 주문을 모아두고,
09:00에 가장 많이 체결될 수 있는 가격으로 시초가를 결정한다.

전일 종가가 300,000원이어도 시초가가 304,000원이 될 수 있다.

이유:
아침에 매수자가 304,000원까지 사겠다고 주문을 넣고,
매도자도 304,000원 이상이면 팔겠다고 주문을 넣으면,
304,000원에서 가장 많은 거래가 성립될 수 있기 때문이다.

정리:
전일 종가 = 어제 마지막 가격
시초가 = 오늘 아침 동시호가 주문으로 새로 정해지는 첫 가격


## 3. 08:30~08:50 구간 주의

08:30~08:50에는 NXT 프리마켓과 KRX 장전 동시호가가 동시에 존재한다.

08:00 ~ 08:50  NXT 프리마켓
08:30 ~ 09:00  KRX 장전 동시호가

같은 08:40 주문이라도 어디로 주문을 넣느냐에 따라 다르다.

NXT 주문:
프리마켓에서 바로 체결 가능

KRX 동시호가 주문:
바로 체결되지 않고 09:00 시초가 결정 때 한꺼번에 처리

따라서 증권앱이나 API에서 주문 시장을 구분해야 한다.

시초가에 참여하려면 KRX 장전 동시호가 주문을 사용해야 한다.
장 시작 전에 바로 체결하고 싶으면 NXT 프리마켓 주문을 사용한다.


## 4. KRX API와 증권사 API 차이

KRX API:
한국거래소에서 제공하는 공식 데이터 API.
주로 일별 시세, 종목 정보, 시장 통계, 지수 정보, 과거 데이터 분석에 적합하다.

증권사 실시간 API:
한국투자증권, 키움증권, LS증권, 대신증권 같은 증권사에서 제공하는 API.
HTS/MTS에서 보는 실시간 시세, 호가, 예상체결가, 주문 기능을 코드로 사용할 수 있다.

정리:
KRX API = 공식 데이터 / 통계 / 일별 데이터 중심
증권사 API = 실시간 시세 / 호가 / 예상체결가 / 주문 중심

장전 동시호가 예상체결가를 보고 자동 매수 전략을 짜려면
KRX API보다 증권사 실시간 API를 사용하는 것이 현실적이다.


## 5. 장전 동시호가에서 확인할 데이터

필요한 데이터:
예상체결가
예상체결량
예상등락률
매수호가
매도호가
호가잔량
거래량

가장 중요한 데이터:
예상체결가 = 현재까지 쌓인 동시호가 주문 기준 예상 시초가
예상체결량 = 그 가격에서 체결될 것으로 예상되는 수량

주의:
예상체결가는 확정 가격이 아니다.
08:59:00에 304,000원이더라도 08:59:50에 307,000원으로 바뀔 수 있다.


## 6. 매수 시작 시나리오 1 - 기본 감시 후 지정가 매수

목적:
장전 동시호가 예상체결가를 계속 확인하다가,
내 기준에 맞으면 09:00 시초가 체결을 노리고 지정가 매수 주문을 넣는다.

호출 시간:
08:50 ~ 08:58
1분마다 예상체결가 / 예상체결량 조회

08:58 ~ 08:59:30
5초마다 예상체결가 / 예상체결량 조회

08:59:20 ~ 08:59:40
조건이 맞으면 KRX 장전 동시호가 지정가 매수 주문 제출

09:00 이후
체결 여부 확인

흐름:
1. 관심 종목 목록 준비
2. 08:50부터 예상체결가 조회 시작
3. 예상체결가, 예상체결량, 등락률 저장
4. 08:58부터 조회 주기를 5초로 줄임
5. 예상체결가 흐름 확인
6. 예상체결가가 내 기준 가격 이하인지 확인
7. 예상체결량이 충분한지 확인
8. 08:59:20~08:59:40 사이 조건 충족 시 지정가 주문
9. 09:00 이후 체결 여부 확인
10. 미체결이면 취소하거나 정규장 전략으로 전환

## 8. 왜 시장가보다 지정가가 안전한가

시장가성 주문을 넣으면 예상보다 높은 가격에 체결될 수 있다.

예를 들어 예상체결가가 304,000원으로 보였는데,
마지막에 매수세가 몰려서 실제 시초가가 307,000원이 될 수 있다.

이때 시장가성 주문이면 307,000원에 체결될 수 있다.

그래서 내 최대 매수가를 정하고 지정가 주문을 넣는 것이 안전하다.

예시:
예상체결가: 304,500원
내 최대 매수가: 305,000원

지정가 305,000원 매수 주문

결과:
시초가 304,500원 → 체결 가능
시초가 305,000원 → 체결 가능
시초가 307,000원 → 체결 안 됨

즉, 너무 비싸게 사는 것을 막을 수 있다.

## 12. 처음 개발할 때 주의

처음부터 실제 주문하지 말고 며칠 동안은 로그만 쌓는 것이 좋다.

며칠 동안 비교할 것:

이걸 보면 종목별로 장전 예상체결가가 얼마나 믿을 만한지 알 수 있다.


## 13. 최종 요약

NXT 프리마켓 가격은 KRX 시초가가 아니다.
KRX 시초가는 08:30~09:00 장전 동시호가 주문으로 결정된다.
08:30~08:50에는 NXT 프리마켓과 KRX 장전 동시호가가 동시에 존재한다.
주문 시장을 구분해야 한다.
장전 예상체결가를 보려면 KRX API보다는 증권사 실시간 API가 현실적이다.
API 호출은 08:50부터 시작하고, 08:58 이후에는 더 자주 호출한다.
주문은 08:59:20~08:59:40 사이에 조건 충족 시 넣는 것이 현실적이다.
시장가보다 지정가 주문이 안전하다.
예상체결가는 확정가가 아니므로 반드시 로그를 쌓고 검증해야 한다.
